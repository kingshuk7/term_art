# term_art

Term-Art is an ASCII Art generator written in bash for Linux Terminal. It's quite fun.

## Usage
```bash
cd ~
git clone git@github.com:kingshuk7/term_art.git
cd term-art
bash term-art ./standard.txt "Hello"
```
The output will be in ASCII Art
```
 _    _          _   _          
| |  | |        | | | |         
| |__| |   ___  | | | |   ___   
|  __  |  / _ \ | | | |  / _ \  
| |  | | |  __/ | | | | | (_) | 
|_|  |_|  \___| |_| |_|  \___/  
                                
                                
```
